# Training Planner

**Автор:** Шайхутдинов Айзат
**Описание:** Простое приложение для планирования и фильтрации тренировок с сохранением данных в JSON.
**Технологии:** Python 3.x, tkinter.

## Возможности:
- Добавление тренировок (дата, тип, длительность).
- Фильтрация по типу и дате.
- Сохранение и загрузка данных из `data/trainings.json`.
- Проверка корректности ввода.

## Примеры использования:
1. Запустите `main.py`.
2. Введите дату (например: 2026-04-30), тип (например: Бег), длительность (например: 45).
3. Нажмите «Добавить тренировку».
4. Для фильтрации введите значения в поля фильтра и нажмите «Применить фильтр».
